# Storyboard

## Scene 1 — Hook / Fragmentation (0–8s)
Visual:
- Neutral/dark background
- Three clean floating panels labeled Forum, Snapshot, Tally
- A single proposal card moves between them
- Thin connector lines animate in
Motion:
- Panels fade/slide in
- Proposal card hops between sources
- Slight camera push-in
Goal:
- Establish fragmented governance journey

## Scene 2 — Pain Point (8–18s)
Visual:
- More proposal cards appear
- Browser-tab or window mockups multiply
- Cursor movement suggests searching manually across platforms
Motion:
- Fast but readable tab switching
- Proposal labels stack and clutter the frame
Goal:
- Show cognitive overload

## Scene 3 — Reframe (18–29s)
Visual:
- One proposal path highlighted: Forum → Snapshot → Tally
- Messy layout collapses into one tidy row
- Status labels appear under the row
Motion:
- Morph multiple sources into one structured tracker row
Goal:
- Turn problem into product category insight

## Scene 4 — Product Reveal (29–38s)
Visual:
- Real product homepage revealed
- Hold on title and KPI cards
- Light zoom on top stats
Motion:
- Clean wipe or zoom reveal
Goal:
- Introduce actual product clearly

## Scene 5 — UI Overview (38–52s)
Visual:
- Screen recording scrolls into the table
- Highlight columns: Status, Offchain Snapshot, Onchain Tally, Last Activity, Links
Motion:
- Subtle crop/zoom callouts over the recording
Goal:
- Show structured visibility

## Scene 6 — Search Flow (52–64s)
Visual:
- Search for `treasury`
- Results narrow down
- Pause on a row showing current status and Forum / Snapshot / Tally links
Motion:
- Search typed in
- Result count updates
- Row glow / outline highlight
Goal:
- Show practical user workflow

## Scene 7 — Close (64–72s)
Visual:
- Clean final composed view of dashboard
- Final text card overlays softly
Motion:
- Background slows, minimal movement
Goal:
- Land the positioning: tracker, not voting app

## Transition notes
- Use wipes/morphs between animated scenes
- Use softer cuts once the real UI begins
- Keep transitions under 500ms unless it is the main product reveal
