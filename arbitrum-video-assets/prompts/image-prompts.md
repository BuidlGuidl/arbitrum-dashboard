# Image Asset Prompts

Use these prompts in ChatGPT image generation. Ask for clean vector-style assets, transparent background when applicable, and consistent visual language.

## 1. Forum / Snapshot / Tally motion panels
Prompt:
"Create three clean UI-style floating panels for a product explainer video, each representing a governance source: Forum, Snapshot, and Tally. Style: premium minimal SaaS, subtle crypto/governance feel, soft shadows, rounded corners, dark-on-light or light-on-dark variants, modern typography, simple abstract content blocks instead of real copyrighted UI. Keep them visually distinct but clearly part of the same design system. 16:9-safe composition, transparent background preferred."

## 2. Traveling proposal card
Prompt:
"Create a single premium proposal card asset for a motion-graphics explainer video. The card should feel like a structured governance item with title, status pill, small metadata rows, and subtle source-link tags. Minimal modern product design, high readability, clean layout, vector-style, transparent background, suitable for animating between different governance stages."

## 3. Governance-flow arrows / connectors
Prompt:
"Create a set of elegant motion-graphics connector assets for a product explainer: arrows, curved connectors, dotted pathways, and flow lines that suggest progression across stages. Minimal, premium, soft glow or subtle contrast, suitable for a SaaS/governance explainer video, transparent background."

## 4. Browser tab clutter mockup
Prompt:
"Create a stylized browser-tab clutter illustration for a product explainer video. Show multiple overlapping browser windows or tabs representing fragmented workflow and information overload. Clean modern vector style, readable but not literal brand copies, premium SaaS aesthetic, slight complexity without chaos, transparent background."

## 5. End-card background illustration
Prompt:
"Create a premium abstract background for a governance/product explainer end card. Style: minimal, elegant, data-product inspired, subtle geometry and soft gradients, no loud crypto clichés, enough negative space for centered text and logo. 16:9 horizontal composition."

## 6. Status tag set
Prompt:
"Create a set of premium UI status pills for a dashboard explainer video: In Discussion, Active Off-chain Vote, Awaiting On-chain Vote, Active On-chain Vote, Executed, Pending Execution. Minimal SaaS style, consistent sizing, clean typography, subtle color differentiation, transparent background."

## 7. Proposal lifecycle strip
Prompt:
"Create a horizontal lifecycle strip for a product explainer showing the stages Forum, Snapshot, and Tally, with simple icons or labels and a clear sense of left-to-right progression. Premium minimal design, modern governance/SaaS aesthetic, transparent background."

## Art direction note
Keep everything:
- minimal
- premium
- not hypey
- more Bloomberg/Figma/Linear than neon crypto poster
