# Voiceover Script

Target length: ~72 seconds

## Full narration
In Arbitrum governance, a single proposal can move from forum discussion to Snapshot to Tally, across different tools and different moments in time.

That makes it harder than it should be to answer one simple question: what stage is this proposal in right now?

The problem is not a lack of information. The problem is fragmentation across the proposal lifecycle.

Arbitrum DAO Governance Tracker brings that lifecycle together into one unified tracker.

You can quickly see whether a proposal is still in discussion, already voted offchain, moving onchain, or fully executed.

Instead of jumping across governance tools, you can search once, scan the current stage, and jump directly to the relevant source.

It is not a place to vote. It is the place to understand what is happening before you do.
